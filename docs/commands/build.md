Builds a C++ project.  
Compiling all source files and linking them into an executable.

## Command

```bash
cf build
```

### Aliases

`build`, `b`

The workspace you run the command in has to contain a .codeframe config file

### Arguments

_Dry Run_  
`--dry-run`  
`-d`

_Verbose_  
`--verbose`  
`-v`

_Mode_  
`--mode`  
`-m`

Values: `debug` `release`
