Creates a new project using one of the available templates.  
Generates a folder with the given name and selected template.

## Command

```bash
cf new
```

### Aliases

`new`, `n`

## Arguments

### Language

`language=` / `lang=` / `l=`

Values:  
`cpp` `c` `asm`

### Template

`template=` / `t=`

Values:  
`default` `cli` `game`
