Creates a new project using one of the available templates.  
Generates a folder with the given name and selected template.

## Command

```bash
cf new
```

### Aliases

`new`, `n`

## Arguments

### Language

`language=` / `lang=` / `l=`

Options:

- `cpp`
- `c`
- `asm` **_( assembly )_**
- `combo` **_( cpp + c + asm )_**

### Template

`template=` / `t=`

Options:

- `default`
- `cli`
- `game`
