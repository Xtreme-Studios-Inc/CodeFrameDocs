Commands

Explore all available CodeFrame commands and what they do.

📄️ Utility Commands

Handy general commands you can use in CodeFrame.

📄️ new

Creates a new project using one of the available templates.

📄️ build

Builds a C++ project.

📄️ bundle

Copies the resources defined in the configuration from the specified "from" path to the "to" path, bundling them together into a single folder.

📄️ run

Run your own commands.

📄️ install

Installs a package from our proof of concept list of packages.

📄️ remove

Removes a single package (dependency) from your project.

📄️ document

COMING SOON: Generate documentation from your code

📄️ execute

COMING SOON: Instantly compiles and runs a script or source file.
